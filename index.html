プライバシーポリシー

最終更新日：2025年1月

本プライバシーポリシーは、「消えるメモ」（以下「本アプリ」）における個人情報の取り扱いについて説明します。

1. 収集する情報

本アプリはユーザーが作成したメモのデータをお客様の端末内にのみ保存します。外部サーバーへのデータ送信は一切行いません。

2. 情報の利用目的

収集したデータはアプリの機能提供のみに使用します。第三者への提供・販売は行いません。

3. App内課金

「1時間見放題チケット」の購入にはAppleのApp内課金機能を使用します。購入情報はAppleによって管理されます。本アプリは購入に関する金融情報を収集・保持しません。

4. Face ID / Touch ID

ロック解除にFace IDまたはTouch IDを使用します。生体認証データは端末内のSecure Enclaveで管理され、本アプリおよびApple以外には送信されません。

5. データの削除

アプリをアンインストールすることで、端末に保存されたすべてのデータが削除されます。

6. 本ポリシーの変更

プライバシーポリシーを変更する場合は、このページに掲載します。

7. お問い合わせ

プライバシーに関するお問い合わせはApp Storeのレビュー欄またはサポートページからご連絡ください。
